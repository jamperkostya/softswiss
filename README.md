# softswiss


Верстка выполнена mobile first.

Разработка велась на less-препроцессоре. На выходе получен скомпилированный минифицированный файл css.

Верстка заняла по времени: 4 часа 15 минут.

Шрифты подобраны "на глаз", так как шрифтов изначально небыло передано и выгружались из сети интернет.

pixel perfect не проверено, потому как свертаны все блоки из задания, но не проверялось на pixel perfect.

При верстке использованы комбинированные применения grid и flex.

Также использованы var и localstorage для смены цветовой гаммы.

Для слайдера использован и кастомизирова swiper.

Также онлайн можно увидеть верстку по ссылке - https://softswiss.artit.com.ua
